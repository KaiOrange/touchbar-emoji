# touchbar-emoji #

**使用touchbar来控制飘落emoji表情雨，for MacOS。**

## 运行源码 🤣 ##

```sh
git clone git@github.com:KaiOrange/touchbar-emoji.git
cd touchbar-emoji
npm install
npm start
```

## 下载最新安装包 😇 ##
